Welcome to the EKU downloader!

EKU's CIS club has put together a Raspberry Pi downloading hub available for all students in BTC 1(??). We have access to many popular software downloads (including Windows 10, Office Suite, Visual Studio) and more for students without access to high speed internet. Aside from these standard software packages, we also offer remote downloads of other software suites, data sets, code repositories and more! These downloads are handled automatically by a codebase built by our very own CIS club members.

Although the downloads are done automatically, this does NOT mean they will be unmonitored. All downloads will be logged by IP address, student ID, and contents, so please refrain from ANY illegal activity!
